Примеры

Пример 1 – переменная, объявленная через let или var внутри функции, не видна за её пределами.

JavaScript
          // через let
function testBlock() {
     let myAge = 28;
     console.log(myAge); // Работает!  28
 }

testBlock(); // вызов функции
console.log(myAge); // Будет ошибка, т.к. переменная не видна за пределами бока

---------

// через var
function testBlock() {
     var myAge = 28;
     console.log(myAge); // Работает!  28
}
testBlock(); // вызов функции
console.log(myAge); // Будет ошибка, т.к. переменная не видна за пределами бока

Пример 2 – переменные, объявленные внутри функции и за её пределами с одним и тем же именем - это 2 разные переменные

JavaScript
          let myAge = 36;

function testBlock() {
     let myAge = 28;
     console.log(myAge); // выведет 28
}
     
testBlock(); // вызов функции

console.log(myAge); // выведет 36
